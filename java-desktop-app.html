Lorenzo il Magnifico Desktop App

Description

A Java desktop application that lets users play a board game online, using a Client-Server approach. Being the rules of the game really complex and the game full of moving parts, it's been designed applying the MVC pattern.

Eclipse was the IDE used to develop the application while Swing has been chosen to create the user interface.
