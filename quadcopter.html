Image Recognition Task on a Quadcopter

Description

As a guest researcher for Tongji University me and an automation engineering student were assigned a project that consisted in making a programmable Quadcopter being able to fly autonomously, following a target chosen remotely, after such target had been categorized (image recognition task) using a Deep Learning algorithm.

To reach my goal and ultimately obtain a working prototype, I extensively used the OpenCV library for Python and experimented a lot using the Caffe framework for Deep learning.

Technology

  • Deep Learning
  • OpenCV
  • Python
  • Caffe
