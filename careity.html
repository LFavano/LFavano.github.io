Careity - Caring Center

Description

A website for a caring center that has been built from scratch and implemented in both its front- and back-end. A Bootstrap template has been used to obtain a better-looking result while the back-end has been developed using NodeJS and implementing the RESTful API design.

The website includes dynamically loaded content that is fetched from different json files. It is currently hosted on Heroku and available at the link provided below.
